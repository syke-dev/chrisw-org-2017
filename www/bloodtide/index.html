Blood Tide is a multiplayer, online, action, real-time strategy game for the PC. The game allows players to experience both the strategic management of an army in command mode and the fast-paced, action-packed combat on the battlefield alongside other troops in field mode.
Two Ways To Play
Tactical-Commander Mode
(RTS)

  • Control three bases
  • Unique units spawn at each base
  • Construct two improvements per base
  • Can spawn a commander and play it
Field-Commander Mode
(Action RPG)

  • Three unique heroes per faction
  • Level up to unlock abilities
  • Can switch back to RTS mode at any time

Blood Tide was created using the C4 engine in C++. Development Time: ~8 months
