Willow

an interactive puzzle game with a strong emphasis on story and emotional impact


Worked on a small group to create a pixel shader to selectively show colors, which allowed us to show only red, green or blue colors (and any combination of).

Took that shader and converted it to a full-screen post processing effect, so it could be applied to the UI and every object in the scene.

Also built the inventory system, puzzles and UI for the game.
